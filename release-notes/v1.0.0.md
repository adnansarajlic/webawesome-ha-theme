# Release v1.0.0 - WebAwesome Initial Release 🚀

This is the very first official release of the WebAwesome theme for Home Assistant!

## 🆕 What's New?
- **Complete Color Palette**: Full implementation of the WebAwesome "Sky" palette for both light and dark modes.
- **Mushroom Optimization**: Built-in variables for transparent backgrounds (`--wa-bg-*`).
- **Global Card-Mod**: Integrated hover effects and modernized interfaces for switches and inputs.
- **HACS Ready**: Fully compliant with HACS directory structure and metadata.

## 🛠️ Technical Details
- Based on Home Assistant 2024.12+ standards for semantic variables.
- Includes automatic color mapping for Light, Switch, and Binary Sensor domains.

Follow the instructions in the README.md for installation via HACS or manual methods.
