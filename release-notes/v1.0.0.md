# Release v1.0.0 - WebAwesome Initial Release 🚀

Detta är den allra första officiella releasen av WebAwesome-temat för Home Assistant!

## 🆕 Vad är nytt?
- **Komplett färgpalett**: Implementerat WebAwesomes "Sky"-palett för både ljust och mörkt läge.
- **Mushroom-optimering**: Inbyggda variabler för transparenta bakgrunder (`--wa-bg-*`).
- **Global Card-Mod**: Inbyggda hovers-effekter och snyggare gränssnitt för switchar och inputs.
- **HACS-redo**: Fullt kompatibel med HACS mappstruktur.

## 🛠️ Tekniska detaljer
- Baserat på Home Assistant 2024.12+ standard för semantiska variabler.
- Inkluderar automatiska färger för Light, Switch och Binary Sensor domäner.

## 📝 Hur man installerar
Följ instruktionerna i README.md för installation via HACS.
