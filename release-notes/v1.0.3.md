# Release v1.0.3

## 笨ｨ Features
* **Static Transparent Chips**: Added new `-static` CSS classes for transparent chips (`wa-chip-success-static`, `wa-chip-info-static`, `wa-chip-warning-static`, `wa-chip-danger-static`). These classes provide the same beautiful transparency as the original chips but without the pulsing ping-animation.
