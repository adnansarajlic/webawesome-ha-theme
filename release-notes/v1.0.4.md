# Release v1.0.4

## ✨ Features
* **Highlight Card Classes**: Added new semantic CSS classes for cards (`wa-card-success`, `wa-card-info`, `wa-card-warning`, `wa-card-danger`). These classes automatically apply a subtle semantic background, a colored icon, and a left border accent to full cards (like Mushroom Template cards), making it easy to highlight specific states without writing custom CSS every time.

## 📚 Documentation
* Updated `README.md` to document the new `wa-card-*` Highlight Card Classes.
* Updated `examples/theme-documentation-card.yaml` to use these new classes instead of hardcoded inline styles.
